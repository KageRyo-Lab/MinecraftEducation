## 使用 Minecraft 教育版內建 Python 編輯器

Minecraft 教育版內建 Python 編輯器，您可以直接在遊戲中使用它來編寫程式。這些程式可以控制遊戲中的 Agent 機器人來執行各種任務。

以下是一個簡單的 Python 程式範例，示範如何讓 Agent 機器人來向前移動一格：  

```python
agent.move("forward")
```

如果要一次使 Agent 機器人向前三格，我們也可以使用迴圈來完成：  

```python
for i in range(3):
    agent.move("forward")
```
